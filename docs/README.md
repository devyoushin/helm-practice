# helm-practice 문서

이 디렉토리는 주제별 번호 체계로 문서를 정리합니다. 앞 번호는 학습과 운영 흐름을 나타내며, 90번대는 저장소 관리용 문서입니다.

## 문서 구조

- [01-installation](./01-installation/): Helm 설치와 업그레이드 - Helm 설치, 업그레이드, 실행 환경 준비 문서를 모읍니다.
- [02-getting-started](./02-getting-started/): Helm 시작하기 - Helm 기본 사용 흐름과 첫 실행에 필요한 문서를 모읍니다.
- [03-core](./03-core/): Helm 핵심 개념 - 차트 구조, 값 파일, 템플릿, 저장소 등 Helm 핵심 개념을 다룹니다.
- [04-advanced](./04-advanced/): Helm 고급 기능 - Hooks, 의존성, 릴리스 생명주기 등 심화 주제를 다룹니다.
- [05-hands-on](./05-hands-on/): Helm 실습 - 직접 차트를 만들고 배포해보는 실습 문서를 모읍니다.
- [06-operations](./06-operations/): Helm 운영 - 운영 팁과 문제 대응에 필요한 문서를 모읍니다.
- [90-standards](./90-standards/): 문서 작성 표준 - 저장소 문서 작성 규칙과 운영 기준을 관리합니다.
- [91-templates](./91-templates/): 문서 템플릿 - 반복해서 사용하는 문서 템플릿을 관리합니다.
- [99-agents](./99-agents/): AI 에이전트 가이드 - 문서 작성과 리뷰에 활용할 에이전트 역할 정의를 관리합니다.
