# Helm Ops

Helm chart와 helmfile 기반 실습 자산을 두는 공간입니다.

| 폴더 | 내용 |
|------|------|
| `install/` | Helm CLI 설치 스크립트 |
| `upgrade/` | Helm release 업그레이드 보조 스크립트 |
| `charts/` | 실습용 Helm chart |
| `apps/` | 애플리케이션 chart 배포 예제 |
| `infra/` | 플랫폼/인프라 chart 배포 예제 |

Chart 원리를 설명하는 문서는 `docs/`에 두고, 실제 chart와 배포 자산은 `ops/`에 둡니다.
